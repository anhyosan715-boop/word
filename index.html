영단어 학습 마스터

체계적인 단어 연습과 집중 학습을 위한 유용한 도구를 제공합니다.

단어 연습 시작하기 학습 타이머 이용하기

앱 소개 및 주요 기능

이 영단어 학습 앱은 사용자가 능동적으로 단어를 익힐 수 있도록 두 가지 핵심 기능을 제공합니다.

  • 단어 연습: 웹사이트를 통해 등록된 단어 리스트를 활용하여 실제 연습 문제를 풀어볼 수 있습니다. 학습 진도를 쉽게 파악하고 반복 학습에 최적화되어 있습니다.
  • 학습 타이머: 집중력을 높이는 데 도움을 주는 타이머 웹 페이지로 이동합니다. 학습 시간을 정확하게 기록하고 관리하여 자기 주도 학습 습관을 형성할 수 있도록 지원합니다.

* 각 버튼을 클릭하면 해당 기능의 전문 웹 페이지로 이동합니다.

안효산

부산기계공업고등학교

이 앱은 사용자가 직접 단어를 등록하고 학습할 수 있도록 설계되었습니다.

바이브코딩 작성소감

바이브코딩 프로젝트를 통해 영단어 학습 앱의 랜딩 페이지를 기획하고 구현하는 과정을 경험할 수 있었습니다. Tailwind CSS를 활용하여 모바일 친화적인 모던한 디자인을 구현하는 데 중점을 두었습니다. 사용자가 쉽게 단어 연습과 학습 도구에 접근할 수 있도록 직관적인 인터페이스를 구성하고자 노력했습니다.
